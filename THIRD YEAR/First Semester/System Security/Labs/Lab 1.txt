6. 
malware - software malitios / creat cu intentia de a cauza rau

virus - malware care se ataseaza altui program si isi inc epe executia odata cu executia acelui program

dropper - componenta ale unui malware care descarca si instaleaza alte componente ale acelui malware 

downloader - ca un dropper, dar mai specializat pe descarcare 

trojan - malware care se prezinta ca un program legitim/util, dar care de fapt e malefic

spyware - software care are scopul de a colecta informatii despre un utilizator fara consimtamantul/cunostinta acestuia

riskware - software care prezinta caracteristici sau commponente care pot fi considerate de risc pentru securitatea sistemului

ransomware - program care cripteaza fisierele unui utilizator si apoi cere o rascumparare pt a le decripta

adware - software care afiseaza anuntuiri nedorite pe ecran

worm - malware care se propaga fara nevoia unei gazde (programe/fisiere existente) sau a interventiei umane

obfuscare - tehnica folosita in programele malware sau in software-ul ilegal pentru a ascunde/ingreuna intelegerea codului sursa al acelui program
