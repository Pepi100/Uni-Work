4. Ce inseamna o functie neglijabila / ne-neglijabila? (Dpdv. al uni adversat PPT)

1. f(x) = 2 => neglijabila
2. f(x) = 1/2000 => neglijabila
3. f(x) = 1/x^2000 => neglijabila
4. f(x) = 1/2^x => neglijabila
5. f(x) = f1(x) + f2(x), unde f1(x) și f2(x) sunt neglijabile => neglijabila
6. f(x) = f1(x) + f2(x), unde f1(x) este neglijabilă și f2(x) este ne-neglijabil => neneglijabla
